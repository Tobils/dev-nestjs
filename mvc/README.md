## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## mvc project
```bash
# create project
$ nest new project

# install express dan template engine
$ npm install hbs dotenv

# generate module user
$ nest g module users

# generate service users
$ nest g service users

# generate controller users
$ nest g controller users
```
